### Lightweight High-Performance Audio Sampler Plugin (JUCE)

> A clean, fast, and modern sampler built for producers who want **zero bloat, instant workflow, and deterministic playback**.

---

## 🚀 Overview

**FreeSampler** is a minimal yet powerful audio sampler plugin built with JUCE, designed to be:

- ⚡ Ultra-lightweight
- 🎯 Deterministic & stable
- 🎛️ Simple but expandable
- 🧠 Built for real-world production workflows

Unlike bloated samplers, FreeSampler focuses on **speed, clarity, and control**.

---

## 🎹 Core Features

- 🎧 Load and play audio samples instantly
- 🔁 Looping support (seamless playback)
- 🎚️ Gain control
- ⏱️ Start / End position trimming
- 🎵 Pitch control (planned / extendable)
- 🧱 JUCE-based plugin architecture (VST3/AU ready)

---

## 🧠 Design Philosophy

FreeSampler follows a **"low-weight, high-control"** philosophy:

- No unnecessary UI clutter
- No hidden processing
- No unpredictable behavior
- Everything is **traceable and deterministic**

## 🛠️ Build Instructions

Requirements:
- CMake 3.15+
- Xcode or Visual Studio
- Internet (JUCE auto-download)

Build:
cmake -B build
cmake --build build
